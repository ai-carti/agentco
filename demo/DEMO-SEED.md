# 🚀 AgentCo Demo Seed Guide

**Цель:** За 2 минуты поднять красивое демо с реальными данными.

---

## Шаг 1 — Создай компанию (30 сек)

1. Открой приложение → нажми **"New Company"** (или кнопку +)
2. Введи название: **`Stealth AI`**
3. Нажми **Create**

---

## Шаг 2 — Добавь агентов (60 сек)

В карточке компании нажми **"Add Agent"** трижды:

| Имя | Роль |
|-----|------|
| **Atlas** | CEO |
| **Nova** | CTO |
| **Pixel** | SWE |

Для каждого: заполни Name + Role → Save.

> Совет: у каждого агента будет уникальный цветной аватар — выглядит сочно.

---

## Шаг 3 — Создай задачу (20 сек)

1. Перейди на вкладку **Board**
2. В колонке **Backlog** нажми **"+ Add Task"**
3. Введи название: **`Build landing page`**
4. Нажми Save

---

## Шаг 4 — Запусти! (10 сек)

1. Кликни по карточке `Build landing page`
2. В открывшейся панели нажми кнопку **▶ Run**
3. Сразу открой вкладку **War Room**

---

## War Room — что показывать инвесторам

- Живой фид агентов (Atlas, Nova, Pixel думают и работают)
- Статусы: 🟢 thinking / running, 🔵 done
- Сообщения агентов в real-time через WebSocket
- Анимации — всё ощущается живым

---

## Финальный чеклист перед демо

- [ ] Backend запущен (`cd repo/backend && uv run uvicorn main:app`)
- [ ] Frontend запущен (`cd repo/frontend && npm run dev`)
- [ ] API key настроен в Settings (OpenAI / Anthropic)
- [ ] Браузер открыт в режиме фокуса (скрыть тулбар)
- [ ] Вкладка War Room открыта и готова
