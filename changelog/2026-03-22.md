## Выполнено Alex 16:30 MSK
- ALEX-TD-080: `SqliteVecStore` — отсутствие `threading.Lock` при shared `sqlite3.Connection` → `OperationalError: database is locked` при concurrent агентах через `run_in_executor`. Фикс: добавлен `self._lock = threading.Lock()`, все DB-операции завёрнуты в `with self._lock:`. `memory/vector_store.py` — fixed
- ALEX-TD-081: `handlers/ws_events.py` — WS subscriber leak при silent TCP disconnect. Клиент закрывает соединение без WS close frame → `queue.get()` блокируется бесконечно → подписчик висит в `_subscribers` навсегда → memory leak. Фикс: concurrent `_watch_disconnect()` task + `asyncio.wait(FIRST_COMPLETED)` — любой disconnect прерывает оба task через `cancel()`. `handlers/ws_events.py` — fixed
