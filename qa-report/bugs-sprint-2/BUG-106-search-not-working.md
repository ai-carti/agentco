# BUG-106 — Кнопка поиска (header) не реагирует

**Priority:** 🟡 Major
**Found by:** @timofeytst

## Симптом
Кнопка поиска в верхней навигации не открывает поиск / не реагирует на клик.

## Вероятная причина
- Компонент не реализован (заглушка без handler)
- onClick не навешен

## Критерий исправления
Клик на поиск → открывается поисковый интерфейс. Минимум: modal/input с возможностью ввода.
