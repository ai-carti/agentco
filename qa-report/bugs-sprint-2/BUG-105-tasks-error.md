# BUG-105 — Нельзя создать/просмотреть задачи компании — ошибка API

**Priority:** 🔴 Critical
**Found by:** @timofeytst

## Симптом
При попытке выбрать компанию и создать задачи — API возвращает ошибку. Задачи не создаются / не отображаются.

## Вероятная причина
- Зависит от BUG-101: компания не создана корректно → tasks endpoint не находит company_id
- Неправильный эндпоинт: /api/v1/companies/{id}/tasks vs /api/v1/tasks
- CORS / Authorization header не передаётся при запросе tasks

## Критерий исправления
В Kanban внутри компании можно создать задачу и она отображается в колонке Backlog.
