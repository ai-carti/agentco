# BUG-102 — War Room: разделы/табы не открываются

**Priority:** 🔴 Critical
**Found by:** @timofeytst

## Симптом
Внутри компании левая навигация / табы War Room не реагируют на клик или показывают пустой контент.

## Вероятная причина
- Роутинг не настроен для sub-path на GitHub Pages (SPA 404 issue)
- Данные не подгружаются из-за бэкенд ошибки (компания не создаётся — см. BUG-101)
- WebSocket соединение падает при отсутствии данных

## Критерий исправления
Клик на War Room / Kanban / Agents внутри компании — контент загружается корректно.
