# BUG-101 — Созданная компания не отображается в списке

**Priority:** 🔴 Critical (блокирует основной флоу)
**Found by:** @timofeytst (ручное тестирование прода)
**URL:** https://ai-carti.github.io/agentco

## Симптом
Пользователь создаёт компанию — запрос уходит, но компания не появляется в списке на UI.

## Вероятная причина
- CORS или API URL проблема: фронт бьёт не на тот эндпоинт
- `/api/v1/companies` vs `/api/v1/companies/` (trailing slash)
- Фронт не рефетчит список после создания

## Критерий исправления
После нажатия "Создать компанию" — компания немедленно появляется в списке без reload страницы.
