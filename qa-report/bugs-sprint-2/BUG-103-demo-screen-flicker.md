# BUG-103 — "Начать демо" — экран появляется на 1 секунду и исчезает

**Priority:** 🟡 Major
**Found by:** @timofeytst

## Симптом
После создания компании при нажатии "Начать демо" (онбординг) — экран агента/War Room появляется на ~1 секунду, затем исчезает. Предположительно происходит при получении ответа от бэкенда.

## Вероятная причина
- Race condition: компонент монтируется, потом unmount при ошибке данных
- После ответа бэкенда происходит redirect обратно на список компаний
- Ошибка в обработке ответа POST /companies → неправильный navigate()

## Критерий исправления
После онбординга пользователь остаётся на экране War Room без мерцания.
