## Что проверить и создать тикеты:

### Базовые вещи (должны быть в любом продукте)
- [ ] **Favicon** — добавить немедленно (КРИТИЧНО)
- [ ] **Title страниц** — каждая страница имеет правильный <title>
- [ ] **Meta description / og tags** — есть хотя бы базовые
- [ ] **404 страница** — что показывается при неизвестном роуте
- [ ] **Loading states** — везде где есть async запросы
- [ ] **Error states** — что показывается при ошибке API
- [ ] **Empty states** — что показывается при пустых данных

### UX
- [ ] **Все кнопки кликабельны** — нет disabled без причины
- [ ] **Все формы валидируются** — required поля, email format
- [ ] **Toast/feedback** при actions (создал → показало success)
- [ ] **Мобильная версия** — хотя бы не сломана
- [ ] **Console.log мусор** — нет в проде

### Производительность
- [ ] **Bundle size** — не раздут
- [ ] **Lazy loading** — тяжёлые компоненты
- [ ] **Изображения** — оптимизированы

### Accessibility
- [ ] **alt тексты** у изображений
- [ ] **aria-labels** у интерактивных элементов без текста
- [ ] **Keyboard navigation** — основные флоу работают с клавиатуры
